# LMS - Front end part

## Getting started
1. `git clone https://github.com/rxyknight/lms.git`
2. `cd lms`
3. `npm install`
4. `npm start`

## Back End

The back end RESTfull API is deployed to Azure, and please check the following Swagger link:

http://lms20180225101429.azurewebsites.net/swagger/ui/index
